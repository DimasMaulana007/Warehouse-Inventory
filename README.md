# Sistem Inventaris IPAK

Aplikasi berbasis PHP native untuk mengelola data bahan baku dan stok barang.

## Cara Menjalankan

1. Clone repo ini
2. Jalankan server lokal (contoh: `php -S localhost:8000`)
3. Import database sendiri (file tidak tersedia untuk keamanan)

## Fitur
- Manajemen bahan baku
- Laporan stok bulanan
